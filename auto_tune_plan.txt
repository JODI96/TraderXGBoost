Plan: auto_tune.py — Deep Analysis + Auto-Tuner                                                                                                                                                                                  │
│                                                                                                                                                                                                                                  │
│ Context                                                                                                                                                                                                                          │
│                                                                                                                                                                                                                                  │
│ The trading system has a trained XGBoost model but relies on several hand-crafted execution filters                                                                                                                              │
│ (d_max_atr, require_squeeze, min_ema9_21_diff, vol_regime) that may be redundant or harmful — the                                                                                                                                │
│ model already saw those features during training. The user wants:                                                                                                                                                                │
│ 1. A single script that deeply analyzes model + backtest performance across 2023, 2024, 2025                                                                                                                                     │
│ 2. Automated parameter tuning (100 runs), saving best model + params                                                                                                                                                             │
│ 3. Feature importance analysis identifying removable features                                                                                                                                                                    │
│ 4. Prediction-only mode: remove all execution filters — only model probabilities drive trades                                                                                                                                    │
│                                                                                                                                                                                                                                  │
│ ---                                                                                                                                                                                                                              │
│ Approach: auto_tune.py (single file, ~700 lines)                                                                                                                                                                                 │
│                                                                                                                                                                                                                                  │
│ Key Design Decision — Prediction-Only Execution                                                                                                                                                                                  │
│                                                                                                                                                                                                                                  │
│ Set these values in every backtest call:                                                                                                                                                                                         │
│ - d_max_atr = 99 (never blocks)                                                                                                                                                                                                  │
│ - require_squeeze = False                                                                                                                                                                                                        │
│ - min_ema9_21 = -99 (never blocks)                                                                                                                                                                                               │
│ - cooldown = 0 (let model decide)                                                                                                                                                                                                │
│                                                                                                                                                                                                                                  │
│ Only tunable parameters: T_up, T_down (probability thresholds) + XGBoost training hyperparams.                                                                                                                                   │
│                                                                                                                                                                                                                                  │
│ ---                                                                                                                                                                                                                              │
│ Phases                                                                                                                                                                                                                           │
│                                                                                                                                                                                                                                  │
│ Phase 1: Setup & Data Loading                                                                                                                                                                                                    │
│                                                                                                                                                                                                                                  │
│ def load_year_data(year: int, cfg: dict) -> tuple[pd.DataFrame, pd.DataFrame]:                                                                                                                                                   │
│     """Load 1m CSV for year, compute features+labels, return (df_feat, df_raw)"""                                                                                                                                                │
│     # Uses: data.load_csv(), features.compute_features(), labels.compute_labels()                                                                                                                                                │
│     # Returns df_feat with label column for backtesting                                                                                                                                                                          │
│ - Load 2023, 2024, 2025 separately (not merged)                                                                                                                                                                                  │
│ - Cache computed features to disk (models/tuning/cache/{year}_feat.pkl) to skip re-computation (~34s/year)                                                                                                                       │
│ - Cache invalidated when config hash changes                                                                                                                                                                                     │
│                                                                                                                                                                                                                                  │
│ Phase 2: Feature Importance Analysis                                                                                                                                                                                             │
│                                                                                                                                                                                                                                  │
│ def analyze_features(model: xgb.Booster, feature_cols: list) -> pd.DataFrame:                                                                                                                                                    │
│     """Return DataFrame with gain, weight, cover per feature, sorted by gain desc"""                                                                                                                                             │
│     # Uses: model.get_score(importance_type='gain')                                                                                                                                                                              │
│     # Also: model.get_score('weight'), model.get_score('cover')                                                                                                                                                                  │
│     # Compute cumulative_gain_pct, flag features with cumulative_pct contribution < 1%                                                                                                                                           │
│ Output to console: table of top-20 features + bottom-20 (removal candidates)                                                                                                                                                     │
│ Save: models/tuning/feature_importance.csv                                                                                                                                                                                       │
│                                                                                                                                                                                                                                  │
│ Phase 3: Year-by-Year Baseline                                                                                                                                                                                                   │
│                                                                                                                                                                                                                                  │
│ def run_year_backtest(df_feat, probs, cfg, T_up, T_down) -> dict:                                                                                                                                                                │
│     """Run backtest with prediction-only filters, return report dict"""                                                                                                                                                          │
│     # Calls run_backtest() with d_max=99, require_squeeze=False, min_ema9_21=-99                                                                                                                                                 │
│ - Load existing model from models/xgb_model.json                                                                                                                                                                                 │
│ - Run on 2023, 2024, 2025 separately                                                                                                                                                                                             │
│ - Print summary table: Year | Trades | WR% | PF | Sharpe | MaxDD | Return%                                                                                                                                                       │
│                                                                                                                                                                                                                                  │
│ Phase 4: Parameter Sensitivity Analysis                                                                                                                                                                                          │
│                                                                                                                                                                                                                                  │
│ def sensitivity_analysis(years_data, model, cfg) -> None:                                                                                                                                                                        │
│     """Sweep each parameter individually, show effect on all 3 years"""                                                                                                                                                          │
│ Parameters swept (trading only, fast):                                                                                                                                                                                           │
│ - T_up / T_down: [0.50, 0.55, 0.60, 0.65, 0.67, 0.70, 0.75, 0.80]                                                                                                                                                                │
│ - (All other execution filters disabled — prediction-only)                                                                                                                                                                       │
│                                                                                                                                                                                                                                  │
│ Saves per-parameter CSV to models/tuning/sensitivity_T.csv                                                                                                                                                                       │
│ Prints table showing how PF/Sharpe changes per year at each threshold value.                                                                                                                                                     │
│                                                                                                                                                                                                                                  │
│ Phase 5: Auto-Tuner (Optuna + random fallback)                                                                                                                                                                                   │
│                                                                                                                                                                                                                                  │
│ def auto_tune(years_data, cfg, n_trials=100, mode='fast') -> dict:                                                                                                                                                               │
│     """                                                                                                                                                                                                                          │
│     mode='fast': only tune T_up, T_down (no retraining, ~seconds/trial)                                                                                                                                                          │
│     mode='full': also tune XGBoost params + retrain each trial (~20 min/trial)                                                                                                                                                   │
│     """                                                                                                                                                                                                                          │
│                                                                                                                                                                                                                                  │
│ Trial execution flow:                                                                                                                                                                                                            │
│                                                                                                                                                                                                                                  │
│ For each trial:                                                                                                                                                                                                                  │
│   1. Sample params:                                                                                                                                                                                                              │
│      Fast:  T_up ∈ [0.50, 0.90], T_down ∈ [0.50, 0.90]  (can be symmetric)                                                                                                                                                       │
│      Full:  + break_weight, max_depth, min_child_weight, subsample,                                                                                                                                                              │
│               colsample_bytree, gamma, reg_alpha, reg_lambda, learning_rate                                                                                                                                                      │
│                                                                                                                                                                                                                                  │
│   2. If full mode:                                                                                                                                                                                                               │
│      a. Mutate deep copy of cfg with sampled XGB params                                                                                                                                                                          │
│      b. Run walk_forward_cv() → best_n_estimators                                                                                                                                                                                │
│      c. Run train_final() → new model                                                                                                                                                                                            │
│                                                                                                                                                                                                                                  │
│   3. For year in [2023, 2024, 2025]:                                                                                                                                                                                             │
│      a. Get probs from model.predict(X_year)                                                                                                                                                                                     │
│      b. run_year_backtest(df_feat, probs, cfg, T_up, T_down)                                                                                                                                                                     │
│      c. If PF < 0.85 → prune trial early (skip remaining years)                                                                                                                                                                  │
│                                                                                                                                                                                                                                  │
│   4. Compute composite score:                                                                                                                                                                                                    │
│      score = (PF_2023 + PF_2024 + PF_2025) / 3                                                                                                                                                                                   │
│              × (Sharpe_2023 + Sharpe_2024 + Sharpe_2025) / 3                                                                                                                                                                     │
│      Penalty: score = 0 if any year has PF < 1.0 OR n_trades < 30                                                                                                                                                                │
│                                                                                                                                                                                                                                  │
│   5. If score > best_score:                                                                                                                                                                                                      │
│      Save model artifacts to models/best_tuned/                                                                                                                                                                                  │
│      Save params to models/tuning/best_params.json                                                                                                                                                                               │
│                                                                                                                                                                                                                                  │
│   6. Append trial record to models/tuning/trial_history.csv                                                                                                                                                                      │
│   7. Print progress every trial: trial N | score | 2023_PF | 2024_PF | 2025_PF | T_up | T_down                                                                                                                                   │
│                                                                                                                                                                                                                                  │
│ Optuna setup:                                                                                                                                                                                                                    │
│ try:                                                                                                                                                                                                                             │
│     import optuna                                                                                                                                                                                                                │
│     optuna.logging.set_verbosity(optuna.logging.WARNING)                                                                                                                                                                         │
│     study = optuna.create_study(direction='maximize',                                                                                                                                                                            │
│         storage=f'sqlite:///models/tuning/optuna.db',  # resumable!                                                                                                                                                              │
│         study_name='btc_scalper',                                                                                                                                                                                                │
│         load_if_exists=True)                                                                                                                                                                                                     │
│     study.optimize(objective, n_trials=n_trials)                                                                                                                                                                                 │
│ except ImportError:                                                                                                                                                                                                              │
│     # Random search fallback — sample uniformly                                                                                                                                                                                  │
│     random_search(objective, n_trials)                                                                                                                                                                                           │
│                                                                                                                                                                                                                                  │
│ Resume support: load_if_exists=True means python auto_tune.py --resume continues from where it left off.                                                                                                                         │
│                                                                                                                                                                                                                                  │
│ Phase 6: Final Report                                                                                                                                                                                                            │
│                                                                                                                                                                                                                                  │
│ def print_final_report(best_params, best_scores, feature_df, trial_history) -> None                                                                                                                                              │
│ Outputs:                                                                                                                                                                                                                         │
│ - Best params table                                                                                                                                                                                                              │
│ - Best year-by-year performance with best params                                                                                                                                                                                 │
│ - Top 20 features to KEEP                                                                                                                                                                                                        │
│ - Bottom N features to REMOVE (cumulative gain < 1%)                                                                                                                                                                             │
│ - Saves models/tuning/analysis_report.txt                                                                                                                                                                                        │
│                                                                                                                                                                                                                                  │

 Phase 1 — Data loading with cache
  - Loads each year separately, computes features+labels, caches to models/tuning/cache/{year}_{confighash}.pkl
  - Auto-invalidated when config.yaml changes

  Phase 2 — Feature importance
  - Extracts gain/weight/cover from the XGBoost model
  - Prints top-20 and removal candidates (gain < 0.5%)
  - Saves models/tuning/feature_importance.csv

  Phase 3 — Year-by-year baseline
  - Runs prediction-only backtest: d_max=99, min_ema9_21=-99, min_vol=0, require_squeeze=False, cooldown=0
  - Prints Year | Trades | WR% | PF | Sharpe | MaxDD% | Ret% table

  Phase 4 — Sensitivity analysis
  - Sweeps T ∈ [0.50..0.80] symmetrically, prints PF/N/Sharpe per year per threshold
  - Saves models/tuning/sensitivity_T.csv

  Phase 5 — Auto-tuner (Optuna + random fallback)
  - fast mode: only T_up/T_down, probs pre-computed once, ~seconds/trial
  - full mode: also tunes XGBoost hyperparams + retrains each trial
  - Early pruning when PF < 0.85 in any year
  - Score = mean_PF × mean_Sharpe (0 if any year fails PF≥1.0 or trades<30)
  - Saves best model to models/best_tuned/, logs all trials to CSV, Optuna DB is resumable

  Phase 6 — Final report saved to models/tuning/analysis_report.txt

  Run commands:
  # Analysis only (fast, no tuning)
  ./trader2/Scripts/python.exe auto_tune.py --analysis-only

  # Fast tuning: 100 trials, ~seconds each
  ./trader2/Scripts/python.exe auto_tune.py --mode fast --trials 100
  ./trader2/Scripts/python.exe auto_tune.py --mode fast --trials 100 --jobs 3

  # Resume interrupted run
  ./trader2/Scripts/python.exe auto_tune.py --resume --trials 100

  # Full tuning: retrain each trial (~20min each)
  ./trader2/Scripts/python.exe auto_tune.py --mode full --trials 20
